# AI Movie Recommendation System

This project is an AI-powered movie recommendation system built using LangChain. It helps users discover movies based on their preferences, viewing history, and natural language queries.

## Project Structure

- **frontend/**: The user interface built with Next.js, allowing users to interact with the recommendation system.
- **backend/**: The Node.js/Express server that handles API requests, integrates with LangChain, and communicates with AI models.



## Technologies Used
- **Frontend:** Next.js, React
- **Backend:** Node.js, Express, TypeScript
- **AI/ML:** LangChain
